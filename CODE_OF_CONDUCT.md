# Code of Conduct

## Our Standard

Participation in this project should be respectful, direct, and constructive.

Examples of acceptable behavior:

- giving technical feedback focused on code and outcomes
- asking clarifying questions before making assumptions
- acknowledging mistakes and correcting them quickly

Examples of unacceptable behavior:

- personal attacks, harassment, or discriminatory language
- hostile or repeated bad-faith arguments
- publishing private information without consent

## Enforcement

Project maintainers are responsible for enforcing this code of conduct.
They may remove, edit, or reject comments, commits, issues, or pull requests
that violate these standards.

## Reporting

If you experience or observe unacceptable behavior, report it privately to the
maintainer. Reports will be reviewed promptly and handled with discretion.
